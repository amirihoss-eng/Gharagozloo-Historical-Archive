# Gharagozloo Historical Archive Explorer — Graph Core

This release adds the Living Historical Graph as the primary opening experience.

## Features
- Expand and collapse family branches at any node
- Parent/child, spouse, and sibling links
- Zoom, pan, fit, and re-center
- Search and center on any person
- Filter visible nodes by archive branch
- Clickable person nodes with a persistent detail drawer
- Clickable parents, children, siblings, and spouses
- Direct access to full person records, claims, and citations

Launch with `launch_explorer.bat`. The application remains read-only and uses the repository root `archive.sqlite`.
