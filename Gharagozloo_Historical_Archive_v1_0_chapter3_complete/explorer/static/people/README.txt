PERSON PORTRAITS
================

Save one portrait in this folder using the person's archive ID as the filename.

Example for Naser al-Molk (P0040):

    P0040.jpg

Supported extensions, checked in this order:

    .webp
    .jpg
    .jpeg
    .png

Use exactly one primary portrait per person. The Explorer automatically displays it
at the top of the right-hand person panel. When no matching file is present, a clean
placeholder appears instead.

Recommended image preparation:
- portrait orientation or a close crop around the person's face;
- at least 800 pixels wide;
- JPG or WebP for photographs;
- keep the original archival image separately outside this display folder.
